HOW TO READ THIS DASHBOARD

ECG WAVEFORM LEGEND

The waveform shape encodes the cybersecurity health of each month:

Normal Sinus Rhythm

SEVERITY 0-40
Strong, regular peaks. Clear P-QRS-T pattern.
= Low infection rates, fast recovery, backups intact. The sector is healthy.

Arrhythmia

SEVERITY 40-65
Irregular rhythm, unstable baseline, variable peaks.
= Rising infections, slower recovery, some backups compromised. The sector is under stress.

Cardiac Arrest

SEVERITY 65-100
Chaotic fibrillation degrading to flatline.
= Severe infections, weeks of downtime, data unrecoverable. The data has flatlined.

ATTACK PATHOLOGY (SANKEY DIAGRAM)

Shows how incidents flow from entry vector through encryption, data theft, backup loss, ransom payment, and recovery outcome.

Red nodes = bad outcome occurred (encrypted, stolen, backups lost, ransom paid)

Green nodes = bad outcome avoided

Click nodes to build a path filter. Select multiple per column. Percentages update to reflect the filtered subset.

INCIDENT EXPLORER

Scatter mode: Each dot is one incident. Choose X and Y axes. Brush to select dots.

Grouped mode: Dots rearrange into categorical columns. White lines show the median. Reveals which defensive postures lead to better outcomes.

Click any dot to see the full incident record.

INTERACTIONS

All three views are bidirectionally linked. Brushing months filters the Sankey and Explorer. Clicking Sankey nodes recomputes the ECG and filters the Explorer. Selecting dots highlights corresponding months and pathways.

The organization type filter (top right) filters all three views simultaneously.
